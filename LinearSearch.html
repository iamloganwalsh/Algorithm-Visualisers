Time Complexity: O(n)

Space Complexity: O(1)

Linear Search is the most basic searching algorithm out there. To perform Linear Search, we want to loop through each element in the array until we find our target number. If we reach the end of the array before finding the target number, we can conclude that our number is not present in the array.

If the array we search is sorted, we can create a fast version of Linear Search where we only check numbers until we find a value greater than our target value. In this case, we know that the array does not contain our number, so we can finish the search without checking the whole array.
