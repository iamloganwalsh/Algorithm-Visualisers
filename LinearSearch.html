Time Complexity: O(n)

Space Complexity: O(1)

Linear Search is the most basic searching algorithm out there. To perform Linear Search, we want to loop through each element in the array until we find our target number. If we reach the end of the array before finding the target number, we can conclude that our number is not present in the array. 


            def LinearSearch(input_array, target_value):

                # Loop through each item in the array
                for index in range(len(input_array) - 1):

                    # Check if the current item is equal to our target value
                    if input_array[index] == target_value:

                        # If we found our item, we return the index where it is stored
                        return index

                # If our item is not in the list, return -1
                return -1
